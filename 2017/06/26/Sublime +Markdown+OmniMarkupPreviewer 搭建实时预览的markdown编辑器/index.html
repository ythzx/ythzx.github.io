Package Install 安装

网络安装

ctrl+反引号打开控制台，在控制台中输入代码

Package 文件安装

preference->Browse Package，然后进入上一个目录，进入Installed Package中

从https://sublime.wbond.Net/Package%20Control.sublime-package下载文件，然后放入上面的目录中

安装插件

输入ctrl+shift+p后输入install,等待网络接通，输入要安装的插件

安装boxy

安装完会让你选择安装A File Icorn

github：https://github.com/ihodev/sublime-boxy

boxy配置文件
