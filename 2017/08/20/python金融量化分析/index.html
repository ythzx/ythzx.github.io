numpy的切片和列表的切片不同

列表的切片是通过copy复制出来的，新复制的修改不会影响原来的值，但是numpy的数组是创建的一个视图，视图修改后，原来的也会修改，这种是针对大数据设计的。可以通过copy解决

b是a的视图

b = a[:4].copy() 这样就是复制的
