tar cjf 直接打包后压缩bzip2

tar xf 是统一的解压方法

3 vi编辑器的使用

进入命令行模式复制改行内容，在下方粘贴80行

yy 80p

G :定位到最后一行

gg:定位到第一行

u:恢复

Ctrl + r：重做命令

通过u恢复后，可以重做

P:上一行粘贴

n dd :删除下面n行

yy:复制当前行

yy p :复制当前行并粘贴

yy np : 复制后向下粘贴n行

H：跳到屏幕的最前面
M：屏幕中间
L：跳到屏幕的最后

n G :定位到第n行

3.1 插入的位置

命令行模式下：

i:在光标的左边插入

a:在光标的右边插入

o:在这一行的下一行插入

O：在这一行的上一行插入

3.2 Esc 输入冒号 进入扩展模式

输入wq退出

输入m 查看帮助信息

  • 输入n新建分区

  • 输入p新建主分区

输入p打印分区的信息

4.2 创建扩展分区

选择e新建扩展分区，将剩余的全部空间都给扩展分区，逻辑分区是在扩展分区上建立的，分区号是从5开始，1-4是主分区

4.5 需要挂载点

挂载点就相当于一条通路，钥匙

首先新建/sdb1 /sdb5 /sdb6 作为挂载点

mount

mount /dev/sdb1 /sdb1

mount /dev/adb6 /sdb6

df -T 查看文件系统类型

df -h 查看挂载点的详细信息

5 软连接 硬链接

5.1　软链接

ln -s /etc/passwd /tmp

软链接是创建了一个文件，这个文件指向原来的文件的名字，原来的文件对应着一个Innode号，innode号指向一个相应的block块，这个block块存放了源文件的内容

软链接可以跨分区建立
